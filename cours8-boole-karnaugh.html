2. Propriétés fondamentales

Associativité :  (a+b)+c = a+(b+c)    (a·b)·c = a·(b·c)
Commutativité :  a+b = b+a              a·b = b·a
Neutre :         0+a = a                1·a = a
Idempotence :    a+a = a                a·a = a
Involution :     ā̄ = a
Complémentarité: a·ā = 0               a+ā = 1
Absorbant :      a+1 = 1               a·0 = 0
De Morgan :      (a+b)̄ = ā·b̄           (a·b)̄ = ā+b̄
Distributivité : a·(b+c) = a·b+a·c     a+(b·c) = (a+b)·(a+c)
Simplification : a+ā·b = a+b           (a+b)·(a+c) = a+b·c

3. Fonctions booléennes

f : {0,1}n → {0,1}. Minterme = produit de toutes les variables (positives ou niées) qui vaut 1 pour une seule combinaison. Maxterme = somme de toutes les variables qui vaut 0 pour une seule combinaison.

FND = somme des mintermes (lignes à 1)
FNC = produit des maxtermes (lignes à 0)

4. Table de Karnaugh

Tableau à double entrée, lignes/colonnes en code de Gray (00, 01, 11, 10). Propriété clé : entre cases adjacentes, une seule variable change.

Règles de regroupement (pour FND) :
1. Regrouper les 1 adjacents par blocs de 1, 2, 4, 8... (puissances de 2)
2. Blocs les plus grands possibles
3. Une case peut appartenir à plusieurs blocs
4. Les bords sont adjacents (table torique !)
5. Chaque bloc → un terme (variables fixes dans le bloc)
Pour la FNC : même méthode mais on regroupe les 0. On obtient la FND de F̄, puis on complémente par De Morgan.

5. Exemple complet

F = ā·b·c̄·d̄ + a·b·c·d + a·b̄·c·d + a·b·c̄·d̄
