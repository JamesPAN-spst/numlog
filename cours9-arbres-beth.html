Arbres de Beth & déduction

C11 — Arbres de vérité, consistance, conséquence logique

1. Principe

Les arbres de Beth décomposent une formule en sous-formules jusqu'aux atomes. Chaque branche ouverte donne une distribution satisfaisante.

2. Règles de décomposition

✓ Astuce : ∧ et ses dérivés (¬∨, ¬→) → même branche. ∨ et ses dérivés (¬∧, →) → branchement. Commencer par les conjonctions pour limiter les branches.

3. Branches fermées et ouvertes

Branche fermée (✗) : contient p ET ¬p pour une même variable → contradictoire.

Arbre fermé : toutes les branches fermées → formule insatisfaisable.

Branche ouverte : donne les valeurs des atomes → modèle satisfaisant.

4. Applications

• Arbre de F → FND (chaque branche ouverte = un terme conjonctif)
• Arbre de ¬F → FNC (via négation de la FND de ¬F)
• Arbre de ¬F fermé → F est une tautologie

5. Conséquence logique (⊨)

∑ ⊨ C signifie : toute distribution satisfaisant ∑ satisfait aussi C.

Méthode : construire l'arbre de {H₁, H₂, ..., Hₙ, ¬C} (tronc = hypothèses + négation conclusion).

Arbre fermé → ∑ ∪ {¬C} inconsistant → déduction légitime
Branche ouverte → contre-exemple existe → déduction NON légitime

6. Consistance

Consistant (satisfaisable) : ∃ au moins une branche ouverte.

Inconsistant : arbre fermé → pas de distribution satisfaisant toutes les formules simultanément.

⚠ Si ∑ est inconsistant : ∑ ⊨ C est TOUJOURS légitime, quelle que soit C. « D'un ensemble contradictoire, on déduit n'importe quoi. »

7. Théorème de la déduction

∑ ∪ {H} ⊨ C   ⟺   ∑ ⊨ (H → C)
Cas ∑ = ∅ :   H ⊨ C   ⟺   ⊨ (H → C)
