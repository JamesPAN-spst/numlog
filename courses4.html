Cours 4 — Virgule flottante

→ Quiz

1. Principe : nombre = ± mantisse × base^exposant

En virgule flottante normalisée, la mantisse commence par 0,d₁d₂...dₖ avec d₁ ≠ 0.

Format 8 bits (exercice type) : B1 | B2B3B4 | B5B6B7B8

ChampBitsRôle
B11Signe (0=+, 1=−)
B2B3B43Exposant
B5B6B7B84Mantisse

2. Version standard (exposant en CA2)

Représenter +7,5 :
  7,5 = (111,1)₂ = 0,1111 × 2³
  Signe: 0
  Exposant 3 en CA2 sur 3 bits: 011
  Mantisse: 1111
  → 0 011 1111

3. Bit caché + exposant biaisé

Bit caché : puisque d₁=1 toujours (normalisation), on ne le stocke pas. On gagne 1 bit de précision.

Exposant biaisé : au lieu du CA2, on ajoute un biais = 2k−1 (k = nombre de bits d'exposant).

Représenter +7,5 avec bit caché et biais :
  7,5 = 0,1111 × 2³
  Signe: 0
  Exposant: 3 + biais(2²=4) = 7 → (111)₂
  Mantisse sans bit caché: 111 0 (on enlève le 1 initial, il reste 1110)
  → 0 111 1110
⚠ Décodage avec bit caché : la mantisse stockée B5B6B7B8 représente 0,1B5B6B7B8 (on remet le 1 devant).
Décoder 1 011 1000 (bit caché + biaisé) :
  Signe: 1 → négatif
  Exposant biaisé: (011)₂ = 3 → exposant réel = 3−4 = −1
  Mantisse: 0,1 1000 (bit caché remis)
  → −0,11 × 2⁻¹ = −(0,011)₂ = −0,375

4. Registre base 10 : S s E M₁M₂M₃...

Format courant en exercice : S=signe, s=signe exposant, E=|exposant|, M=mantisse avec M₁≠0.

Registre 8 cases : S s E M₁M₂M₃M₄M₅
Exemple : 0 1 3 56789 → +0,56789 × 10⁻³

Bornes

Plus petit positif : 0,10000 × 10⁻⁹   (M₁=1 min, exposant = −9)
Plus grand positif : 0,99999 × 10⁹    (M max, exposant = +9)

Nombre de valeurs représentables

S: 2  ×  sE: 19 (−9 à +9)  ×  M₁: 9 (1 à 9)  ×  M₂...M₅: 10⁴
= 2 × 19 × 9 × 10⁴ = 342 000

5. Précision machine ε

ε = plus petit nombre tel que 1 + ε ≠ 1 dans la représentation.

Registre S s E M₁M₂M₃M₄M₅ :
  1 = 0,10000 × 10¹
  Successeur = 0,10001 × 10¹
  ε = 0,00001 × 10¹ = 10⁻⁴
ℹ Formule : Pour une mantisse de k chiffres en base B : ε = B1−k. Ici B=10, k=5, donc ε = 10⁻⁴.

6. Pièges : overflow et perte de précision

2 + 3 = 5 en flottants 3 bits mantisse base 2 :
  2 = 0,100 × 2² ;  3 = 0,110 × 2²
  Somme = 1,010 × 2² = 0,101 × 2³ → si exposant max = 2 : OVERFLOW
Perte d'associativité :
  a=−7, b=7, c=0.001 (registre 2 chiffres mantisse base 10)
  (a+b)+c = 0 + 0,001 = 0,001 ✓
  a+(b+c) = −7 + 7,001 → 7,001 arrondi à 7,0 → a+7,0 = 0 ✗
