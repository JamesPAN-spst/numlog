Cours 3 — Complément à deux (CA2)

→ Quiz

1. Principe

Sur n bits, le CA2 représente les entiers de −2n−1 à 2n−1−1. Sur 8 bits : −128 à +127.

Le bit de poids fort (MSB) indique le signe : 0 = positif, 1 = négatif.

2. Coder un nombre positif

+44 sur 8 bits : 44 = 32+8+4 → 0010 1100

3. Coder un nombre négatif

−63 sur 8 bits :
  1. Coder +63 :     0011 1111
  2. Inverser bits :  1100 0000
  3. Ajouter 1 :     1100 0001  ← CA2(−63)
✓ Vérification : 0011 1111 + 1100 0001 = 1|0000 0000 (retenue ignorée = 0). C'est correct.

4. Décoder un CA2

1001 1001 :
  MSB=1 → négatif
  Inverser: 0110 0110, +1: 0110 0111 = 103
  → Nombre = −103

5. Addition / Soustraction

A − B se calcule comme A + CA2(−B). On additionne normalement, on ignore la retenue sortante.

127 − 44 :
  0111 1111    (+127)
+ 1101 0100    (−44)
───────────
1|0101 0011 → ignorer retenue → 0101 0011 = 83 ✓

6. Overflow (dépassement)

⚠ Règle : Overflow ssi on additionne deux nombres de même signe et le résultat a un signe différent.
Équivalent : overflow ssi bit de signe du résultat ≠ retenue vers le bit de signe.
63 + 65 :     0011 1111 + 0100 0001 = 1000 0000
  Deux positifs → résultat négatif → OVERFLOW !
70 − 71 :     signes différents → JAMAIS d'overflow
ℹ Interprétation mathématique : A − B se calcule comme A + (2n − B) − 2n. Le CA2(−B) = 2n − B, et la suppression de la retenue correspond au −2n.
